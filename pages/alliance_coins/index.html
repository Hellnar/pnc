Alliance help

Helping alliance members is the main way to collect alliance coins. The game give rewards for 50 helps per day. Each help will give you 1,000 coins

Alliance donations

Donations for alliance research can give you additional coins every day. Each donation will give you at least 250 coins. But there is a random chance that game will give you a better reward.

Alliance trial

In alliance trial you can also get alliance coins. You need to set an assist hero. This hero will get some coins just for being availabla for assistance for other alliance member. But when someone will pick your hero and add to their lineup, you will receive additional coins.

Alliance convoy

Every day alliance can send one convoy. There are 7 slots members can hold. Each slot will give some amount of coins. It's always good if convoy leader will re-roll the rewards for better prizes or for more coins.

Table shows how convoy coins are shared for all members

Position Coins
Leader50k + sum of all passenger coins
DeputySum of passenger coins
PassengerOnly coins from their own cart

Example:

Leader: 50k + 0 + 10k + 25k + 50k

Deputy 1: 0 + 10k + 25k + 50k

Deputy 2: 0 + 10k + 25k + 50k

Passenger 1: 0 coins

Passenger 2: 10k coins

Passenger 3: 25k coins

Passenger 4: 50k coins

Infernal gates
